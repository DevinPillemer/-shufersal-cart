API ENDPOINT DETAILS
================================================================================

Base URL: /api/shufersal
Method: POST
Content-Type: application/json

Supported Actions:
1. search - Search for products
   Input: {"action":"search","query":"milk"}
   
2. add-to-cart - Add item to cart
   Input: {"action":"add-to-cart","query":"cheese"}
   
3. get-cart - Get cart contents
   Input: {"action":"get-cart"}

Response Format:
{
  "success": boolean,
  "status": "SUCCEEDED|FAILED|ABORTED",
  "action": "search|add-to-cart|get-cart",
  "query": "user query",
  "results": [...],
  "message": "status message"
}
